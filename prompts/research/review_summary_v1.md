{{ customer_reviews }}

## 任务

综合所有评论，提炼出结构化的评论摘要，供后续生成本品属性表使用。要求：

1. 严格基于评论内容，不得编造；评论未提及的内容不要补充
2. 尽量保留买家的真实用词（口语化表达），便于后续文案借用
3. 若评论带有 ASIN 来源，按 ASIN 归类评分与好评/差评点；无来源则归入整体

## 输出格式

只返回一个 JSON 对象：

```json
{
  "pros": ["高频好评点1", "高频好评点2"],
  "cons": ["高频差评点1", "高频差评点2"],
  "high_frequency_issues": ["反复出现的质量/使用问题"],
  "usage_scenarios": ["评论中提到的真实使用场景"],
  "user_language": ["买家常用的原话短语"],
  "by_asin": {
    "B0XXXXXXXX": {
      "rating": 4.2,
      "pros": ["该竞品好评点"],
      "cons": ["该竞品差评点"]
    }
  }
}
```

## 说明

- 每个数组尽量给出 3 条以上（数据不足时如实给出更少）
- by_asin 仅在评论可识别 ASIN 来源时填写，否则返回空对象 {}
- rating 为该 ASIN 评论的平均分（0-5），无法判断时省略该字段
