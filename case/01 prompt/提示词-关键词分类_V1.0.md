## 主提示词

```
## 任务说明

你将收到以下两类输入：
1. 一份产品属性表（结构化文本）
2. 一份从欧鹭工具中导出的ASIN流量词库（表格数据）

你的任务是：
- 基于产品属性表，建立完整的产品语义地图
- 基于语义地图和词库数据，将每个关键词分类到对应类别
- 输出结构化的分类结果，供运营团队直接使用

---

## 输入格式说明

### 产品属性表字段（你将收到以下信息）
- 产品名称（中文+英文，可能包含多个别称）
- 材质
- 颜色/规格
- 适用人群/场景
- 核心功能
- 打动买家的点
- 不适用场景（重要：用于判断否定词）

### 欧鹭词库字段说明
你将收到的字段及其含义如下：
- 关键词：待分类的搜索词
- 月搜索量：买家每月在亚马逊搜索该词的次数
- 月搜索排名：该词在所有词中的搜索量排名
- 日均搜索量：月搜索量/30
- 建议出价：竞价参考均值（美元）
- 最小建议出价 / 最大建议出价：竞价区间
- 标题覆盖度：竞品Listing标题中包含该词的比例
- 在售产品数：亚马逊上与该词相关的在售商品数量
- 机会指数：平台评估该词的投放机会评分（满分10分）
- 自然搜索重合数 / 广告搜索重合数：与自然/广告结果重合的ASIN数
- 搜索转化率：搜索该词后发生购买行为的比例
- 月销量：通过该词带来的估算月销量
- 点击份额(%) / 转化份额(%)：Top ASIN在该词下的流量和转化集中度

| 材质/成分词 | |

| 使用场景词 | |
| 症状/问题描述词 | |

### 3. 语义边界说明

## 第二步：关键词分类

基于语义地图和词库数据，将每个关键词分配到以下四个类别之一。

---

### 分类体系与判断规则

#### 🔴 A类——流量核心词
**定义：** 类目大词，搜索量大，代表买家的泛需求入口，是品牌曝光和类目竞争的主战场。

**判断标准（同时满足）：**
- 语义：词义宽泛，代表整个品类或核心需求，通常为1-3个词的短词
- 月搜索量 ≥ 5,000
- 标题覆盖度 ≥ 30%（说明这是行业公认的核心词）

**新品期使用建议：** 用于Listing标题埋词（自然流量布局），广告投放时搭配低竞价宽泛匹配，以曝光为主，不以转化为核心目标。

---

#### 🟠 B类——精准转化词
**定义：** 精准描述本产品具体特征、功能或使用场景的词，买家搜索意图明确，转化率高，是新品期广告投放的核心词组。

**判断标准（同时满足）：**
- 语义：包含具体功能、材质、人群、场景等限定词，词义精准指向本产品
- 月搜索量 ≥ 500
- 搜索转化率 ≥ 5%
- 不触发否定条件（见D类规则）

**新品期使用建议：** 广告精准匹配优先投放，同时埋入Listing五点和产品描述中。这类词ROI最高，是新品期预算的核心投放对象。

---

#### 🟡 C类——场景长尾词
**定义：** 描述具体使用场景、特定人群或细分需求的长尾词，搜索量偏小但意图明确，竞争度低，适合低成本获取精准流量。

**判断标准（同时满足）：**
- 语义：包含场景、人群、特定问题描述，通常为3个词以上的长词组
- 100 ≤ 月搜索量 < 5,000
- 搜索转化率 ≥ 3%
- 在售产品数 ≤ 50,000

**新品期使用建议：** 广告词组匹配或精准匹配，竞价可低于B类词。适合作为广告自动投放的补充，或用于A+页面和QA埋词。

---

#### ⚫ D类——否定词
**定义：** 语义不相关，或语义相关但数据表现差、投放性价比极低的词。新品期投放这类词会直接浪费预算。

**触发否定的条件（满足任意一条即归入D类）：**

*语义层面（优先判断）：*
- 词义明显指向本产品不适用的场景或人群（参考语义地图中的"语义边界"）
- 词义指向其他品类，与本产品无关联
- 词义包含本产品明确标注的禁忌场景（如：nasal disorders相关词）
- 看起来像竞品品牌名或其他品牌专有词（包括人名、品牌词、型号词等非通用词汇）

*数据层面（语义相关但数据差）：*
- 月搜索量 < 100
- 搜索转化率 < 2%
- 在售产品数 > 50,000 且 月搜索量 < 1,000
- 机会指数 < 3
- 建议出价 > $3.0 且 搜索转化率 < 5%（新品期高竞价低转化，不值得投）

**新品期使用建议：** 在广告活动中添加为否定关键词，阻止广告匹配到这些词，减少无效曝光和无效点击消耗。

## 第三步：输出分类结果

### 输出格式要求

先输出语义地图（第一步），再输出分类结果表格。

分类结果表格格式如下：

| 关键词 | 类别 | 月搜索量 | 搜索转化率 | 建议出价($) | 机会指数 | 分类依据说明 | 推荐使用位置 |
|--------|------|---------|-----------|------------|---------|------------|------------|

**推荐使用位置说明（按类别固定填写）：**
- 🔴 A类：Listing标题 / 广告宽泛匹配（曝光用）
- 🟠 B类：广告精准匹配（重点投放）/ Listing五点+描述埋词
- 🟡 C类：广告词组匹配（补量用）/ A+页面+QA埋词
- ⚫ D类：广告否定关键词

**分类依据说明要求：**
- A类：注明"大词/类目核心词，月搜索量[X]，标题覆盖度[X]%"
- B类：注明主要命中的精准特征，如"精准功能词，转化率[X]%"
- C类：注明场景或人群特征，如"场景长尾词，搜索量[X]，竞争度低"
- D类：注明否定原因，语义原因写"语义不相关：[具体原因]"，数据原因写"数据表现差：[具体触发条件]"，品牌词写"疑似品牌词：[判断依据]"

---

### 输出结束后附加以下汇总信息

**词库分类汇总：**
- 总词数：X个
- 🔴 A类（流量核心词）：X个（占比X%）
- 🟠 B类（精准转化词）：X个（占比X%）
- 🟡 C类（场景长尾词）：X个（占比X%）
- ⚫ D类（否定词）：X个（占比X%）

**新品期广告投放优先级建议：**
1. 第一优先级：B类词全部纳入精准匹配广告组
2. 第二优先级：C类词中搜索转化率最高的前20个纳入词组匹配广告组
3. 第三优先级：A类词选取标题覆盖度最高的前5个做宽泛匹配曝光组
4. 立即执行：D类词全部添加为广告活动否定词

---
## 注意事项

1. 每个词只能归入一个类别，不允许重复分类
2. 当一个词同时满足多个类别条件时，按以下优先级判断：D类 > B类 > C类 > A类（否定优先，精准优先）
3. 数据字段缺失时，优先依赖语义判断，并在分类依据中注明"[X字段数据缺失，依据语义判断]"
4. 不要对词库数据做任何修改，原样保留关键词文本
5. 如遇明显的拼写错误或乱码词，单独列出，标注"数据异常，建议人工核查"
```
